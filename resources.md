## SQL and Databases

- [SQL Performance Explained](https://use-the-index-luke.com/) - A comprehensive guide to understanding SQL performance and query tuning.
- [SQLZoo](https://sqlzoo.net/wiki/SQL_Tutorial) - Interactive SQL tutorials and problem sets.
- [Database Star SQL Cheat Sheet](https://www.databasestar.com/sql-cheat-sheet/) - SQL keyword cheat sheet.
- [SQLBolt](https://sqlbolt.com/) - Free online interactive SQL tutorial.
- [Modern SQL](https://modern-sql.com/) - Advanced SQL topics by the author of 'SQL Performance Explained'.
- [Learn SQL](https://www.databasestar.com/learn-sql/) - A massive collection of over 60 SQL learning resources.
- [JOOQ SQL Joins Blog](https://blog.jooq.org/say-no-to-venn-diagrams-when-explaining-joins/) - Detailed explanation of SQL joins and why Venn diagrams fall short.
- [Advanced SQL by Great Learning](https://www.mygreatlearning.com/academy/learn-for-free/courses/advanced-sql) - Free video course covering common advanced SQL concepts.
- [Kaggle SQL and BigQuery Tutorial](https://www.kaggle.com/code/dansbecker/getting-started-with-sql-and-bigquery/tutorial) - A practical guide on integrating and querying datasets using Google's BigQuery.
- [SQL Practice](https://skillsverification.co.uk/sql-practice/) - Environment to practice syntax and table operations in SQL.
- [Advanced SQL Puzzles](https://advancedsqlpuzzles.com/) - Tricky logic and data manipulation problems to test advanced SQL skills.
- [Analyst Builder](https://www.analystbuilder.com/) - A platform catered toward data analysts for improving SQL and Python skills.
- [8 Week SQL Challenge](https://8weeksqlchallenge.com/) - Case study-based exercises to boost SQL portfolio projects and analytical thinking.
- [Practice Window Functions](https://www.practicewindowfunctions.com/) - A targeted site dedicated to mastering SQL window functions with interactive exercises.
- [DataLemur SQL Questions](https://datalemur.com/sql-interview-questions) - SQL practice and interview questions.
- [DataLemur SQL Environment](https://datalemur.com/) - General platform for data science and SQL practice.
- [StrataScratch](https://www.stratascratch.com/?via=jenifer&gad_source=1&gad_campaignid=21401633031&gclid=Cj0KCQjwlYHBBhD9ARIsALRu09rQf8kUBadq9OcMKhOe5JjwI5YwZFihfdO3nEIbN5cMo8__lMlSs84aAqjEEALw_wcB) - Platform for data science and SQL interview preparation featuring real company questions.

## Programming Languages

### Python

- [Automate the Boring Stuff](https://automatetheboringstuff.com/) - Online tutorial and book for learning Python for non-developers.
- [Automate the Boring Stuff Udemy Discount](https://inventwithpython.com/blog/automate-boring-stuff-udemy-free-discount.html) - Blog post from the author on where to get his book for the best price.
- [CS50P](https://www.edx.org/learn/python/harvard-university-cs50-s-introduction-to-programming-with-python) - Harvard University's rigorous introduction to programming with Python.
- [FutureCoder](https://futurecoder.io/) - Interactive path to learning Python from scratch.
- [MOOC.fi Python Programming](https://programming-25.mooc.fi/) - The University of Helsinki's legendary, thorough free Python course.
- [Python 3 Library Reference](https://docs.python.org/3/library/index.html) - Official standard library documentation for Python 3.
- [Python 3.13 Documentation](https://docs.python.org/3.13/index.html) - Latest official documentation for the newest Python release.
- [Python 3.13 Tutorial](https://docs.python.org/3.13/tutorial/index.html) - Official guided tutorial straight from the Python Software Foundation.
- [Python Roadmap](https://roadmap.sh/python) - Step-by-step visual learning path detailing what to learn as a Python developer.
- [FutureCoder IDE](https://futurecoder.io/course/#ide) - Direct link to the web-based IDE used in the FutureCoder Python curriculum.

### C and C++

- [LearnCpp](https://learncpp.com/) - Online tutorial for C++ starting from the very basics and going into great depth.
- [VS Code Clang Config](https://code.visualstudio.com/docs/cpp/config-clang-mac) - Official VS Code doc for configuring the Clang C/C++ compiler on a Mac.
- [Tutorialspoint C++ Compiler](https://www.tutorialspoint.com/compilers/online-cpp-compiler.htm) - Fast, web-based C++ compiler.
- [Wandbox](https://wandbox.org/) - Web-based compiler comparison for C and C++.
- [Godbolt Compiler Explorer](https://godbolt.org/) - C and C++ compiler that also outputs machine code instruction sets for different CPU architectures.
- [C++ Standards](https://www.open-std.org/jtc1/sc22/wg21/docs/standards) - Final drafts of official C++ standards. The published versions are paid, but these final drafts are free and close enough.

### Go

- [Golang Resources Thread](https://www.reddit.com/r/golang/comments/n5ppx5/some_resources_that_have_helped_me_learn_golang/) - Reddit collection of Go learning resources.

- [Gophercises](https://gophercises.com/) - Free coding course that walks through building mini applications in Go.
